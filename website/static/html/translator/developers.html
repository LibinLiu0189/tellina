Tellina translates natural language to bash commands. For example, it translates "Find text file in the current folder" into the bash command find . -name "*.txt".

How to use Tellina

Translating a command

Type a complete imperative sentence in the search page and click the "Translate" button to tranlate it to a bash command.. If Tellina isn't helpful, you can also search your sentence on Google using the "Search with Google" button.

Interpreting the results

The result page lists multiple possible translations.

Tokens highlighted in orange are command heads, tokens highlighted in purple are flags of a command, and tokens with underline are arguments or flags.

Not every translation may be correct. . You might need to modify the command as well, especially the underlined arguments.

If you don't understand a command, you can hover over each part of the command for an explanation (from bash man pages). The gear icon on the right takes you to explainshell.com which gives similar functionality.

You can copy a command to your clipboard by clicking the button on the right hand side.

Implementation

Tellina is written in Python and uses bashlex, Tensorflow and Django. Source code is available on github.
